RLayout::Importers

Home of Importers. Importers generate nodes for the virtual file system that are capable of reading content from various sources.

Import file from an HTTP/HTTPS server.

Supported Options

Besides options of HttpFileNode::new the following options are supported.

  • :node_name - Function that takes an http URI and converts it to a node name.

lfs_glob(pattern, opts = {}) click to toggle source

Import all local files matching globbing pattern pattern.

If pattern contains a recursive globbing expression, this method creates sub-nodes and assigns files accordingly. Sub-nodes are created for all directory chains found after the first recursive globbing expression (**).

Supported Options

See options of LFSGlobbingHelper::new

Example 

 Importers.lfs_glob('rlayout/**/e*.rb')

 Importers.lfs_glob('rlayout/**/e*.rb', :node_name => Proc.new {|path| File.basename(path) + '.bak')
