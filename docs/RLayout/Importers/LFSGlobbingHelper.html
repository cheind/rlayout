RLayout::Importers::LFSGlobbingHelper

Supports enumerating files matching a globbing pattern.

Supported Options

  • :node_name - Function that takes a file path and converts it to a node name.

  • :flatten_recursive - If true, recursive patterns do not trigger the creation of virtual sub-nodes.
